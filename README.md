# geo-project
흉기난동 예고 맵 시각화 프로젝트

# How to use
1. knifemap 홈페이지에서 흉기난도 예고 알림 지도를 확인할 수 있음(http://knifemap.cafe24app.com/)
2. 하단 표에서 데이터를 클릭하면 해당 데이터의 좌표로 지도를 줌인 함
3. 제보하기에서 흉기난동 예고 혹은 지난 정보 삭제를 요청함
4. 지도에서 마커를 클릭하면 세부 내용을 확인할 수 있음
